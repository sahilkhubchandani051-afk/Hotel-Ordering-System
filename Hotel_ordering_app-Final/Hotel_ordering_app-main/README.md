# Hotel Ordering System

A modern web-based ordering system for hotels, built with FastAPI and designed for easy deployment.

## 🚀 Quick Start

### Local Development
1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```
2. **Run the Application**
   ```bash
   python run_app.py
   ```

### Deploy to Render.com
See [START_HERE.md](START_HERE.md) for detailed deployment instructions.

**Repository**: `https://github.com/omkardhamane637-lang/Hotel_ordering_app.git`

## 🔧 Configuration
### Environment Variables
Configure these in your Render dashboard:
- `SESSION_SECRET_KEY`
- `ADMIN_EMAIL`
- `ADMIN_PASSWORD`
- `ENVIRONMENT=production`
