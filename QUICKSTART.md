# 🚀 Quick Start - Deploy to Render in 5 Minutes

**Repository**: `https://github.com/omkardhamane637-lang/Hotel_ordering_app.git`

## Step 1: Push Code to GitHub

```bash
git remote add origin https://github.com/omkardhamane637-lang/Hotel_ordering_app.git
git branch -M main
git add .
git commit -m "Initial commit - Render ready"
git push -u origin main
```

## Step 2: Deploy on Render

1. Go to https://dashboard.render.com/
2. Click **"New +"** → **"Blueprint"**
3. Connect your repository: `Hotel_ordering_app`
4. Click **"Apply"**
5. Wait 2-5 minutes ⏱️

## Step 3: Access Your App! 🎉

Your app will be live at:
`https://hotel-ordering-app-XXXX.onrender.com`

**Login:**
- Email: `admin@hotel.com` (default)
- Password: `admin123` (default)

---

**That's it! You're live on Render! 🚀**
